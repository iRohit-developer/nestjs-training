# API Endpoints

Base URL: http://localhost:3000

## Product APIs

1. GET /product
2. GET /product/:id
3. GET /product/expensive/:minPrice
4. GET /product/search/title?title=<text>
5. GET /product/search/category?category=<categoryName>
6. GET /product/search/price-range?minPrice=<min>&maxPrice=<max>
